
# WTWR (What to Wear)

## About the Project
**WTWR** is a React-based application designed to help users decide what to wear based on the current weather in their location. The app fetches real-time weather data and suggests appropriate clothing items from a curated list, ensuring you're always dressed comfortably for the conditions.

## Features
- **Real-time Weather:** Fetches current temperature and weather conditions using the OpenWeather API.
- **Smart Suggestions:** Recommends outfits based on the current temperature.
- **Dynamic Backgrounds:** The interface changes between day and night themes depending on the local time and weather.
- **Interactive Modals:** Users can view detailed information about clothing items through pop-ups.

## Technologies Used
- **React**: For building the user interface and managing component state.
- **Vite**: For a fast development environment and optimized production builds.
- **OpenWeather API**: To retrieve live meteorological data.
- **CSS (BEM methodology)**: For structured, maintainable styling.
